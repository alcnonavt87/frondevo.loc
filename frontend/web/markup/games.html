Наша реализация HTML5 игр дает возможность быстрой интеграции с любой back end платформой.

Используемые технологии

В зависимости от задачи, мы используем разные подходы:

  • HTML5 Canvas API + Javascript
  • HTML5 + CSS3 + Javascript
  • HTML5 + SVG + CSS3 + Javascript
Оптимизация производительности

Для положительной реакции аудитории, анимация должна работать безупречно. Для этого мы используем ряд техник и ухищрений, чтобы получить высокую производительность на разных браузерах и платформах.

Возможность back end разработки

Так же мы можем предложить разработать back end часть для HTML5 игры. Для разных задач применяем разные варианты исполнения:

  • PHP5 / Yii2 + AJAX
  • PHP5 / Yii2 + Laravel
  • NodeJs
